# Login, Registration, and Forgot Password Pages

This project contains three HTML pages: Login, Registration, and Forgot Password, styled with CSS and enhanced with JavaScript for form validation and background effects.

## Files

- `login.html`: The login page
- `register.html`: The registration page
- `forgot.html`: The forgot password page
- `styles.css`: CSS styles including animated gradient background
- `script.js`: JavaScript for form validation

## Features

- **Animated Background**: A smooth gradient animation that shifts colors.
- **Form Validation**: Client-side validation for required fields.
- **Responsive Design**: Centered forms with semi-transparent backgrounds.

## Usage

1. Open any of the HTML files in a web browser (e.g., `login.html`).
2. Fill in the forms and submit to see validation in action.
3. Links between pages allow navigation.

## Troubleshooting

- Ensure all files are in the same directory.
- If JavaScript doesn't work, check browser console for errors.
- For a live server, use VS Code's Live Server extension or similar.

## Notes

This is a front-end only implementation. For real authentication, integrate with a backend server.
